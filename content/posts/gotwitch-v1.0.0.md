GoTwitch v1.0.0 is out! It's a simple web app that allows you to pick a random streamer using a set of filters. It's built with Go and Vue.js.

## Release v1.0.0

### What's new 

- Filter by game
- Filter by language
- Save filters on local storage
- Random streamer picker
